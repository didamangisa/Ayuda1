# KA0 : MİNİMUM DÜŞEY KURPLAR (UZUN K.) : Formüllere göre :                         #
#----------------------------------------------------------------------------------#
# İÇBÜKEY DÜŞEY KURP: Kv= (Dp*Dp) / ( 2 * ( HF - H2 + Dp*Tg(ALF)) )                   #
# DIŞBÜKEY DÜŞEY KURP: KV= (Dp*Dp) / ( 2 * (sqrt(H1) + sqrt(H2))^2 )                   #
# Burada Dp = (V * Tp/3.6) + (V*V / (254*(Fl + i)) )                               #
# Hıza göre tablodan Fl alınarak ve i=0 yapılarak (Yönetmeliğe göre)         #
#----------------------------------------------------------------------------------#
  KA0U   1           : 0->Bu formülleri kullanma  1->Bu formülleri kullan           #
####################################################################################
# KA1 : MİNİMUM DÜŞEY KURPLAR (KISA K.) : UZUNLUK Kontrolü  L >=  KA1 * Vp      #
#----------------------------------------------------------------------------------#
# Eğer düşey kurp uzunluğu KA1 * V'den küçükse, KV bu değeri sağlayana kadar artırılır:   #
# KV = L / A = KA1 * Vp/ A   burada:                                            #
# KA1: yapılandırılabilir bir katsayı, İspanyol yönetmeliğine göre 1.00                   #
#   A: Eğim farkının mutlak değeri, ondalık olarak                              #
#  Vp: Proje Hızı                                                               #
# 5.3.2.2 Sayfa 65
#----------------------------------------------------------------------------------#
  KA1U   1           : 0->Bu formülleri kullanma  1->Bu formülleri kullan           #
  KA1    1.00        :   1.00 İspanyol Yönetmeliğine göre                                #
####################################################################################
# KA2 : MİNİMUM DÜŞEY KURPLAR /KISA K.) : UZUNLUK Kontrolü > Dp                #
#----------------------------------------------------------------------------------#
# Eğer düşey kurp uzunluğu Dp'den küçükse, kısa kurplar için formüller kullanılır:        #
# kısa kurplar:                                                                 #
# İÇBÜKEY DÜŞEY KURP: KV= 2*Dp/A + (2 * ( HF - H2 + Dp*Tg(ALF)) ) / (A*A)             #
# DIŞBÜKEY DÜŞEY KURP: KV= 2*Dp/A + (2 * (sqrt(H1) + sqrt(H2))^2 ) / (A*A)             #
# Burada:                                                                         #
#   A: Eğim farkının mutlak değeri, ondalık olarak                              #
#----------------------------------------------------------------------------------#
  KA2U   1           : 0->Bu formülleri kullanma  1->Bu formülleri kullan           #
