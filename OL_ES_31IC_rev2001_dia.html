# DURUŞ GÖRÜŞ MESAFESİ:  Dp = (V * Tp/3.6) + (V*V / (254*(Fl + i)) )                #
# Tabloda i=0 kullanılmıştır   (kırmızı kot eğimi, ondalık olarak)     #
#----------------------------------------------------------------------------------#
# İSTENEN DEĞERLER: Bir hız artışı için hesaplanmıştır  #
# 20 Km/s                                                                       #
#----------------------------------------------------------------------------------#
# DERE KURBU: Kv= (Dp*Dp) / ( 2 * ( HF - H2 + Dp*Tg(ALF)) )                   #
# TEPE KURBU: KV= (Dp*Dp) / ( 2 * (sqrt(H1) + sqrt(H2))^2 )                   #
#----------------------------------------------------------------------------------#
# GEÇİŞ İÇİN DÜŞEY KURP : Sadece Tepe Kurpları ve Tek Yönlü Platformlar İçin      #
# Geçiş Görüş Mesafesi kullanılır ve formülde yerine konur           #
# Engelin H2 değeri 1.1 ile değiştirilir                                                         #
####################################################################################
# KA0 : MİNİMUM DÜŞEY KURPLAR (UZUN K.) : FORMÜLLERE GÖRE :                         #
#----------------------------------------------------------------------------------#
# DERE KURBU: Kv= (Dp*Dp) / ( 2 * ( HF - H2 + Dp*Tg(ALF)) )                   #
# TEPE KURBU: KV= (Dp*Dp) / ( 2 * (sqrt(H1) + sqrt(H2))^2 )                   #
# Dp = (V * Tp/3.6) + (V*V / (254*(Fl + i)) ) olmak üzere                               #
# Fl tablodan hıza göre alınarak ve i=0 yapılarak (Yönetmeliğe göre)         #
#----------------------------------------------------------------------------------#
  KA0U   1           : 0->Bu formülleri kullanma  1->Bu formülleri kullan           #
####################################################################################
# KA1 : MİNİMUM DÜŞEY KURPLAR (KISA K.) : UZUNLUK KONTROLÜ  L >=  KA1 * Vp      #
#----------------------------------------------------------------------------------#
# Eğer düşey kurp uzunluğu KA1 * V'den küçükse, bu değeri sağlayana kadar KV artırılır:   #
# KV = L / A = KA1 * Vp/ A   olmak üzere:                          #
# KA1: yapılandırılabilir bir katsayı, İspanyol yönetmeliğine göre 1.00                   #
#   A: Eğimler arasındaki farkın mutlak değeri, ondalık olarak             #
#  Vp: Proje hızı                                                       #
#----------------------------------------------------------------------------------#
  KA1U   1           : 0->Bu formülleri kullanma  1->Bu formülleri kullan           #
  KA1    1.00        :   1.00 İspanyol Yönetmeliğine göre                                 #
####################################################################################
# KA2 : MİNİMUM DÜŞEY KURPLAR /KISA K.) : UZUNLUK KONTROLÜ > Dp  (AASHTO Yönetmeliği)#
# (İspanyol Yönetmeliğinde kullanılmamaktadır)
#----------------------------------------------------------------------------------#
# Eğer düşey kurp uzunluğu Dp'den küçükse, kısa düşey kurplar için formüller kullanılır:        #
#                                                                 #
# DERE KURBU: KV= 2*Dp/A + (2 * ( HF - H2 + Dp*Tg(ALF)) ) / (A*A)             #
# TEPE KURBU: KV= 2*Dp/A + (2 * (sqrt(H1) + sqrt(H2))^2 ) / (A*A)             #
# Olmak üzere:                                                                         #
#   A: Eğimler arasındaki farkın mutlak değeri, ondalık olarak             #
#----------------------------------------------------------------------------------#
  KA2U   0           : 0->Bu formülleri kullanma  1->Bu formülleri kullan           #
