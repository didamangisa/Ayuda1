# Yönetmelik dışı tasarımları düzeltir; ancak yer yoksa ...                #
#                                                                              #
# Geçiş eğrisi var; ancak ipmax için kısaysa, dever rampası kurba doğru          #
# uzatılmaya çalışılır, geçiş eğrisi-dairesel kurp birleşim noktasında         #
# deverin dPipmax'tan fazla düşürülmediği kontrol edilir:                     #
#  0.- Kenar eğimini hesaplar ve ipmax'tan büyükse, kurp girişinde   #
#      dPipmax kadar azaltılmış bir deverle kenar eğimi etüdünü gerçekleştirir                                                  #
#  1.1.- Kenar eğimi hala ipmax'tan büyükse, deveri   #
#        ipmax değerine kadar düşürür. Rampa geçiş eğrisi içinde bırakılır ve   #
#        deverin düşürüldüğüne dair bir rapor notu düşer.                               #
#  1.2.- Kenar eğimi ipmax'tan büyükse, toplam deverin ipmax rampası ile   #
#        dairesel kurbun 1/dLipmax'ına kadarını kullanarak uygulanıp uygulanamayacağı etüt edilir.    #
#        Uygulanabiliyorsa, dairesel kurptan gereken kısmı alır.           #
#  2.1.- Dairesel kurbun 1/dLipmax'ını almasına rağmen hala ipmax aşılıyorsa, ve         #
#        dNipmax==0 ise, o zaman deveri düşürür ve bir rapor verir              #
#  2.2.- dNipmax==1 ise, dever rampasını 1/dLipmax'a kadar,  #
#        ipmax aşılsa bile deveri düşürmeden, uygulanabilen ip ile uzatır#
#  2.3.- dNipmax==2 ise, rampayı dairesel kurbun 1/dLipmax'ına kadar uzatır  #
#        ve sığıyorsa geri kalan gerekli kısmı aliymandan alarak ipmax'ı sağlamaya çalışır.    #
#                                                                  #
#  2.4.- Sığmıyorsa, dairesel kurbun 1/dLipmax'ını ve aliymanın mevcut kısmını kullanır.      #
#        Kenar eğim değişim oranı ipmax'tan büyük olur; ancak #
#        deveri düşürmez                                                               #
#  3.- Aliyman kısa ise ve çatı eğimi geçişleri çakışıyorsa,  #
#      dever rampasının başlangıcını aliymanın ortasına çeker ve      #
#      dairesel kurba taşan rampa uzunluğunun bir kısmını geri alır     #
#  4. -  Eki 2013(d) Kurplar ve geçiş eğrileri kısa olduğunda,  #
#        ipmax (kenar eğimi) ve kurptaki Lmcur sabit dever boyu aynı anda sağlanamaz.  #
#        dNipmax=2 ile, ipmax ve Lmcur'u sağlamak için dever düşürülür.  #
#        dNipmax=3 ile dever düşürülmez, ipmax'ı sağlamak için Lmcur düşürülür ve son çare olarak  #
#        ipmax da aşılır.                                           #
#                                                                              #

# dNipmax  2        # 2:ipmax ve  deveri koru ve kalanı aliymandan al    #

# 2/8 #  Deverin çatı eğiminden küçük olduğu kesimlerin maksimum limiti               #
####### ---------------------------------------------------------------------- #
# Kırmızı kot eğimi < mras ise, maksimum çatı eğimi geçiş boyu.                #
# Kurp-aliyman / geçiş eğrisi-aliyman çatı eğimi geçişini belirtir         #
# Ayrıca, büyük geçiş eğrileri durumunda, suyun birikebileceği kesimleri  #
# kısaltmak için çatı eğimine eşit bir dever verisi eklenir               #
#                                                                              #
# Deverin çatı eğiminden küçük olduğu kesim Mlbo'dan uzunsa, #
# Lbo metreye çatı eğimi değerinde bir dever verisi eklenir.                            #
# Bu uzunluk, sıfır deverden çatı eğimi değerine kadar ölçülür.             #
# -%2'den +%2'ye geçiş en fazla 2*Mlbo metrede yapılır.                         #
#                                                                              #
# Sadece kırmızı kot eğimi mras'tan küçükse analiz edilir             #
# mras = 0.0 ise kırmızı kota bakılmaksızın her zaman kontrol eder.                         #
#                                                                              #

# 4/8 #  Geçiş eğrisiz kesimlerde dever rampası kontrolü             #
#######  Geçiş eğrisinin olmadığı durumlar için bir dever rampası uzunluğu  #
#        belirtilir ve aliyman ile kurptaki kısımları tanımlanır   #
#        Tanımlanan uzunluk, program tarafından kurptaki deverden  #
#        trbo'nun tanımladığı değere geçişi yapmak için kullanılır.                                                         #
#        Bu değer 3 farklı şekilde hesaplanabilir :                 #
#                                                                              #
# Eğer rampa aliymana sığıyorsa, tamamını aliymana koy; sığmıyorsa       #
# Rmtr ve Cmtr'yi kullan                                                         #
# RmtrSC                                                                       #
# A.- Maksimum kenar eğim değişim oranı değerine göre               #
# (Daha önce bir değerler tablosuna göre belirtilen IPmax)                #
  RmtrIP  1.0 : Rmtr=K*Pe*NC*AC*ACfac/ipmax   Aliymandaki veya büyük Pe'li kurptaki oran  #
  CmtrIP  0.0 : Cmtr=K*Pe*NC*AC*ACfac/ipmax   Kurptaki veya küçük Pe'li kurptaki oran  #
# ----- --------  ------------------------------------------------------------ #
# B.- Bir dever değişim hızı kullanarak:                          #
#                                                                              #
# RmtrK   0.04   : Rmtr = 0.04 * Pe * V     Aliymandaki veya büyük Pe'li kurptaki oran #
# CmtrK   0.04   : Cmtr = 0.04 * Pe * V     Kurptaki veya küçük Pe'li kurptaki oran #
# ----- --------  ------------------------------------------------------------ #
# C.- Formüle göre hesaplanmış sabit bir değer kullanarak                             #
#     uzunluk =   L0r  + L1r * kurp_deveri                                            #
# ------   ----  ----                                                          #
#  Rmtr   15.0   0.0           Aliymandaki oran (veya daha düşük deverli kurptaki)    #
#  Cmtr    0.0   0.0           Kurptaki oran (veya daha yüksek deverli kurptaki)    #
# ------   ----  ----                                                          #
#                                                                              #
# Pe = mutlak değerde KURP deveri - sıfır dever farkı                   #
# AC = Nominal platform genişliği (genişletmeler hariç)                              #
# Kullanılan formülden bağımsız olarak, hesaplanan değer her zaman     #
# daha sonra kullanabilmeniz için Rmtr ve Cmtr değişkenlerinde saklanır.   #
#                                                                              #
