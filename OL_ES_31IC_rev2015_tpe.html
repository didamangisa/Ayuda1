#        girip girmediği incelenir, dairesel kurbun 1/dLipmax'ına kadar kullanılarak.    #
#        Eğer sığıyorsa, dairesel kurptan gereken kısım alınır.           #
#  2.1.- Dairesel kurbun 1/dLipmax'ını almasına rağmen ipmax aşılırsa ve         #
#        dNipmax==0 ise, dever azaltılır ve bir rapor verilir.              #
#  2.2.- Eğer dNipmax==1 ise, dever geçişi, ipmax aşılsa bile giren ip ile  #
#        1/dLipmax'a kadar uzatılır, ancak dever düşürülmez.#
#  2.3.- Eğer dNipmax==2 ise, geçiş dairesel kurbun 1/dLipmax'ına kadar uzatılır  #
#        ve sığıyorsa geri kalan gerekli kısmı aliymandan alarak ipmax'ı sağlamaya çalışır.    #
#                                                                 #
#  2.4.- Eğer sığmazsa, dairesel kurbun 1/dLipmax'ını ve aliymanın mevcut kısmını kullanır.      #
#        Kenar eğimi ipmax'tan büyüktür; ancak deveri düşürmez.                                                               #

# 2/8 #  Çatı eğiminden daha az deverli kesimlerin maksimum sınırı               #
####### ---------------------------------------------------------------------- #
# Düşey güzergah eğimi < mras ise, maksimum çatı eğimi geçiş uzunluğu.                #
# Kurp-aliyman / klotoit-aliyman çatı eğimi geçişini belirtir         #
# Ayrıca, büyük klotoitler durumunda, suyun birikebileceği kesimleri kısaltmak için #
# çatı eğimine eşit bir dever verisi eklenir.               #
#                                                                              #
# Eğer çatı eğiminden daha az deverli kesim Mlbo'dan daha uzunsa, #
# Lbo metre mesafeye çatı eğimi değerinde bir dever verisi eklenir.                            #
# Bu uzunluk, dever 0 (sıfır) noktasından çatı eğimi değerine kadar ölçülür.             #
# -%2'den +%2'ye geçiş maksimum 2*Mlbo metrede yapılır.                         #
#                                                                              #
# Sadece düşey güzergah eğimi mras'tan düşükse analiz edilir.             #
# Eğer mras = 0.0 ise, düşey güzergahtan bağımsız olarak her zaman kontrol eder.                         #
#                                                                              #
# 3.1 IC 2015 4.7.2 Sayfa 54                                                     #
#                                                                              #
#       Mlbo  mras  Lbo                                                        #
#------ ----  ----  ---- ----------------------------------------------------- #

#######  Para los casos en los que no hay clotoide se especifica una longitud  #
#        de transición de peralte, definiendose la parte en recta y en curva   #
#        La longitud que se define es utilizada por el programa para realizar  #
#        la transicion desde el peralte de la curva hasta el valor que define  #
#        trbo.                                                                 #
#        Puede calcularse este valor de 3 formas diferentes  :                 #
#                                                                              #
# Si la transicion cabe en la recta ponerla toda en la recta; si no cabe       #
# utilizar Rmtr y Cmtr                                                         #
# RmtrSC                                                                       #
# A.- En función del valor de máxima pendiente relativa de borde               #
# (IPmax especificada anteriormente según una tabla de valores)                #
  RmtrIP  1.0 : Rmtr=K*Pe*NC*AC*ACfac/ipmax   Frac. en recta o curva mayor Pe  #
  CmtrIP  0.0 : Cmtr=K*Pe*NC*AC*ACfac/ipmax   Frac. en curva o curva menor Pe  #
# ----- --------  ------------------------------------------------------------ #
# B.- Usando una velocidad de variación del peralte :                          #
#                                                                              #
# RmtrK   0.04   : Rmtr = 0.04 * Pe * V     Fraccion en recta o curva mayor Pe #
# CmtrK   0.04   : Cmtr = 0.04 * Pe * V     Fraccion en curva o curva menor Pe #
# ----- --------  ------------------------------------------------------------ #
# C.- Usando un valor fijo calculado según formula                             #
#     long =   L0r  + L1r * per_cur                                            #
# ------   ----  ----                                                          #
#  Rmtr   15.0   0.0           Fraccion en recta (o curva de menor peralte)    #
#  Cmtr    0.0   0.0           Fraccion en curva (o curva de mayor peralte)    #
# ------   ----  ----                                                          #
#                                                                              #
# Pe = dif de peralte CURVA - peralte cero en valor absoluto                   #
# AC = Ancho de calzada nominal (sin sobreanchos)                              #
# Independientemente de la formula usada, el valor calculado siempre queda     #
# guardado en las variables Rmtr y Cmtr para que lo pueda usar más adelante.   #
#                                                                              #
