Posición

La información de la pestaña Posición y de la pestaña Plataformas se almacena en archivos *.wtg. Existen las siguientes opciones para la gestión de dichos archivos:

  • : Inicia el menú.
  • : Cargar archivo *.wtg.
  • : Añade archivo *.wtg.
  • : Guarda archivo *.wtg.
La creación de posiciones de aerogeneradores (= WTG) puede ser manual o a partir de colecciones de posiciones.

Al crear posiciones de aerogeneradores, se toma como tipo de aerogenerador el tipo 1 de la pestaña Tipos. Se recomienda previamente tener al menos un tipo de aerogenerador creado antes de crear las posiciones, ya que al dibujar considera estos tipos.

Para la creación manual de las posiciones de aerogeneradores, existen la siguientes opciones:

  • : Añade una posición de aerogenerador.
  • : Elimina la posición del aerogenerador.

Para la asignación de las coordenadas X, Y, se puede realizar en Modo Clic o en Modo Numérico, Para la asignación en Modo Clic, se pulsa sobre el botón y posteriormente en pantalla para tomar la X,Y.

Para la creación de posiciones de aerogeneradores a partir de colecciones, existen las siguientes posibilidades:

  • : Crea las posiciones a parir de símbolos.
  • : Crea las posiciones importando un archivo *.csv.

Desde el botón   se guardar un archivo *.csv con las posiciones actuales, dicho archivo puede ser modificado y vuelto a cargar, contemplando así los cambios realizados.

Para la asignación de la Z, tanto en modo manual como a partir de colecciones de aerogeneradores, se actúa de la siguiente manera:

  • Al crear las posiciones de los aerogeneradores se emplea un tipo de aerogenerador definido en la pestaña Tipos.
  • Cada tipo tiene un radio de cálculo de la Z mínima. Con este radio, al crear las posiciones, se calcula la Z mínima que corresponde a la Z en la que no hay terraplén para el radio indicado.
  • La Z mínima calculada se asigna de forma automática a la Z techo y Z explanada, empleando el redondeo de la configuración .
  • Posteriormente las Z techo y la Z explanada se pueden cambiar manualmente.
  • Estas Zs pasan a la Pestaña Plataformas, donde pueden adaptarse para cada subexplanada.
  • Si se pulsa sobre el icono , se copia la Z mínima, teniendo el cuenta el redondeo definido en configuración .


Existe una configuración a la que se tiene acceso desde el botón  con las siguientes opciones:

  • Redondeo cotas: Desplegable en el que se puede elegir el redondeo de Z techo y Z explanada (a partir de la Z mínima).
  • [ ] Generar listados de cimentación: Si está activo, generar listados de cimentación de forma automática. Si está desactivo, se puede generar listados de cimentación desde el botón . La cimentación se define en la pestaña Tipos.
  • [ ] Cimentación / líneas de excavación: Si está activo, se dibuja la cimentación y líneas de excavación. La cimentación se define en la pestaña Tipos.
  • [ ] Líneas de frontera: Si está activo, se dibujan las líneas de frontera. Las líneas de frontera se definen en la pestaña Tipos.
  • [ ] Líneas de perfil: Si está activo, se dibujan las líneas para los perfiles de la pestaña Perfil. Estas líneas para los perfiles se definen en la pestaña Tipos.
  • [ ] Ver todos los nombres de WTG: Si está activo, se visualizan todos los nombres de los aerogeneradores (WTG). Si está desactivo, se visualiza únicamente el nombre del WTG actual.
  • [ ] Ver todas las explanadas: Si está activo, se dibujan todas las explanadas de montaje de los aerogeneradores. Si está desactivo, se dibuja únicamente la actual.
  • [ ] Líneas de firmes: Si está activo, se dibujan las líneas de firme definidas en la pestaña Tipos.
  • Nombre base:  Nombre base empleado para los archivos *.wtg y *.exp. Si no existe se emplea un nombre base por defecto.
  • Archivo *.exp: Nombre del archivo *.exp asociado al *.wtg.

Una vez creadas las posiciones de los aerogeneradores se pueden dibujar las explanadas de montaje de cada posición. Para ello, se tienen en cuenta las siguientes consideraciones:

  • Se tiene en cuenta la configuración definida .
  • Se dibujarán las explanadas de montaje de los aerogeneradores con los tipos asignados (archivos *.tge). Estos tipos deben estar creados previamente en la pestaña Tipos.
  • Se dibujarán las explanadas de montaje que estén activas en el listado de posiciones de aerogeneradores.


 
  • A la hora de dibujar, se emplean las Zs y las pendientes definidas en la pestaña Plataformas. Si en la pestaña Plataformas no se ha modificado nada, las Zs utilizadas coinciden con las definidas en la pestaña Posición.
  • Para dibujar pulsar en .
Para borrar las explanadas de montaje, se pueden emplear los iconos . También se pueden borrar desde procedencias .


Para desplazarnos de unas posiciones de aerogeneradores a otras, se pueden emplear las siguientes opciones:

  • : Se desplaza al primer aerogenerador de la lista.
  • : Se desplaza al último aerogenerador de la lista.
  • : Se desplaza al aerogenerador anterior.
  • : Se desplaza al aerogenerador siguiente.
  • : Se puede seleccionar picando un aerogenerador para desplazarse a él.
  • Casilla Actual: Se puede teclear el aerogenerador al que se desea desplazar.
  • : Zoom al aerogenerador.

A cada explanada de montaje de aerogeneradores se le pueden realizar las siguientes acciones:

  • Azimut: Se pude definir numéricamente o bien con en el botón  para modificar interactivamente en pantalla.
  • Lado: Se puede seleccionar hacia qué lado (derecho / izquierdo) realizar una simetría del tipo asignado (pestaña Tipos).
  • Posición: Se puede seleccionar hacia dónde se quiere hacer una simetría (Inicio / Final) según el tipo asignado (pestaña Tipos).
  • Eje asociado: Se puede asociar un eje a cada posición, bien tecleando el número, pulsando en el botón Buscar, o seleccionando en pantalla (picando).

Tipos

En esta pestaña se crean tipos de distribuciones de plataformas de montaje de aerogeneradores. Cada tipo se almacena en archivo de extensión *.tge, se dispone de botones de guardado  y cargado . Al guardar el archivo *.wtg se guardan también todos los *.tge de la pestaña Tipos. Se puede crear una batería de tipos (= batería de archivos *.tge) que se emplearán en cada posición de aerogenerador, en la pestaña Posición.

Cada tipo contiene la información definida en esta pestaña Tipos (geometrías de las explanadas de montaje y acopio, líneas fronteras, líneas para los perfiles, número de firme a emplear en el menú Explanadas, variables de cimentación, etc).

Para la gestión de los tipos existen las siguientes opciones:
  • : Añade un tipo.
  • : Inserta un tipo.
  • : inserta una copia del tipo.
  • : Borra un tipo.

Cada tipo tiene la posibilidad de definir geometría de la cimentación, para ello hay que activar la casilla  contigua al botón "Cimentación" . Las variables que definen las geometría de la cimentación son las indicadas en la imagen siguiente:




Variables de cimentación:

  • Pedestal: Se define por radio y canto.
  • Losa: Se define por radio y canto.
  • Tronco: Se puede dar por Altura o por Pendiente.
  • Hormigón de limpieza: Se ha de activar la casilla para su uso. Se define por espesor y radio.
  • Excavación: Se define por el sobreancho de la excavación y talud.
  • Sobreexcavación: Se ha de activar la casilla para su uso. Se define por profundidad, radios y talud.
  • Punto de cálculo para la Z min: Con este punto, se calcula el Radio para el cálculo de la Z min. Existen varias opciones de puntos, incluso un valor personalizado.
La configuración las variables del menú cimentación puede guardase y recuperarse en archivos *.cme


La definición de las distribuciones de montaje de aerogeneradores pueden ser definidas en modo Paramétrico o en modo Vectorial. En modo paramétrico  se disponen de las siguientes variables:



Variables en modo paramétrico:

  • Aerogenerador:
    • Radio de cálculo de Z min: si se emplea definición de geometría de cimentación es calculado según los valores indicados en el menú cimentación. si no se emplea cimentación, se indica valor numérico para el cálculo de la Z mínima.
    • Distancia al eje: Distancia del aerogenerador al eje del vial.
    • Expla aero / nacelle: Si se activa se define por longitud y ancho.
    • Ancho del eje: Ancho del vial.
  • Explanada montaje:
    • Distancia a WTG: Distancia de esta subexplanada al aerogenerador.
    • Long. interior: Longitud interior.
    • Long. exterior: Longitud exterior.
    • Ancho: Ancho de esta subexplanada.
    • Explanada Extra: Se puede tener una explanada de acopio extra, al otro lado del vial, el usuario define el ancho.
  • Acopio de palas:
    • Distancia a WTG: Distancia de esta subexplanada al aerogenerador.
    • Longitud: Longitud.
    • Ancho: Ancho.
  • Herramientas / Casetas: Se ha de activar (casilla de activación) para su uso.
    • C1: Subexplanada Caseta1, por Longitud y Ancho.
    • C2: Subexplanada Caseta 2 ,por Longitud y Ancho.
    • Longitud: Longitud.
    • Ancho: Ancho.
  • LFR, perfiles y firmes : Accede al modo Vectorial para la definición de Líneas Frontera, Líneas de Perfil y Firmes.
  • Grúas Auxiliares:
    • Nº grúas: Número de posiciones para la grúa auxiliar de montaje de la pluma. Para cada posición de grúa auxiliar se pueden definir ancho, longitud interior y exterior, distancias inicio, fin e intermedia; así como el lado en el que se ubican.
    • Tipo pluma: Se puede elegir si la subexplanada de la pluma se solapa o no con las de las grúas auxiliares.
    • Ancho berma / Ancho Pluma: Ancho según se tenga un Tipo pluma u otro.
    • Longitud Total: Se activa cuando el Nº de grúas es 0 para dar la longitud. En caso contrario es calculado.

Para la definición en modo vectorial  se conmuta el desplegable a "Vectorial". Al conmutar al modo vectorial, permanece en la pestaña Tipos las variables Distancia al eje (distancia del aerogenerador al eje del vial) y el Ancho Eje (Ancho del vial) que ya existían en modo paramétrico.

El botón abre un entorno gráfico, similar a otras herramientas de Istram. Los tipos (estilos) de líneas a emplear para cada subexplanada son los indicados en la siguiente imagen (L3250, L2351, L2352, etc):



Para facilitar la labor de asignación de los tipos de líneas se puede emplear los botones del cuadro de diálogo. Haciendo clic en los botones de cada tipo (estilo) y luego gráficamente sobre las líneas, se asignan los tipos (estilos) automáticamente.

Los vectores de las subexplanadas han de cumplir las siguientes condiciones:

  • No debe haber puntos repetidos.
  • Los vectores de subexplanadas contiguas no se deben cruzar unos con otros.

Los distintos tipos (estilos) de líneas tendrán los siguientes comportamientos especiales:

  • Los estilos de líneas L2350, L2351, L2352, L2353, L2354 y L2355:
    • Serán posteriormente empleados como línea Coronación de la explanada (L53) del menú de explanadas.
    • Tienen asociado, en este cuadro de diálogo, un número de Firme (en la columna Firme) que será utilizado como el firme a emplear en el menú explanadas.
  • Estilo Líneas de Frontera (L2356): Las líneas de este estilo serán posteriormente empleadas como Línea de frontera (L308) en el menú de explanadas.
  • Estilo Perfiles (L2357): Las  líneas de este estilo serán empleadas posteriormente en la pestaña Perfiles de Parques Eólicos para generar archivos *.per
  • Estilo Firmes (L616):
    • Las líneas de este estilo serán posteriormente empleadas como Líneas de firmes (L616) en el menú de explanadas.
    • Estas líneas serán líneas interiores a otras subexplanadas. Si se quiere que una línea de firme pase por algún segmento de una subexplanada (vaya por el borde) es necesario dejar un pequeño espacio entre la línea de la subexplanada y la línea de firmes (dejar la línea de firmes por el interior de la subexplanada).
    • Actualmente se admiten 2 líneas interiores de firme por subexplanada.

Tanto en modo Paramétrico como en modo Vectorial se puede gestionar las zonas (subexplanadas) de las explanadas de montaje del aerogenerador, accediendo mediante el botón , el cual da acceso al siguiente menú:



En este menú se pueden agrupar Zonas (subexplanadas), por ejemplo se puede unir las zonas del aerogenerador con la zona de montaje y/o con la zona de la pluma. O bien unir las zona del acopio con las de las casetas. Para ello se pasan las zonas de la columna derecha a la columna izquierda con las flechas de la columna central;  posteriormente se selecciona "Zonas sin asignar" y se pasan las zonas a unir a la columna derecha con las fechas al tipo deseado.

En la pestaña Plataformas, se pueden modificar las Zs y las pendientes de cada una de las subexplanadas que conforman el área de montaje de los aerogeneradores.

En esta pestaña hay un desplegable para seleccionar la posición de aerogenerador a modificar . Además se puede seleccionar picando en pantalla y hacer Zoom al aerogenerador actual .

Para la modificación de las Zs y de las pendientes del aerogenerador actual se puede actuar en los controles indicados en la siguiente imagen:




Para modificar los valores, tener en cuenta las siguientes consideraciones:

  • Se actúa sobre el aerogenerador actual.
  • Dentro del aerogenerador actual se pueden seleccionar las distintas subexplanadas con los botones disponibles , para ser modificadas. De tal forma que la subexplanada que se está modificando:
    • Se marca con el icono .
    • En planta se resalta.
    • En el visor 3D se resalta, se muestran las cotas y el punto de origen.
    • Los valores a modificar de la subexplanada actual son:
      • Pend. Longitudinal: Pendiente longitudinal aplicada en al punto de origen.
      • Pend. Transversal: Pendiente transversal aplicada en el punto de origen.
      • Cota: Z de todos los vértices de esta subexplanada.
  • Para cambiar el punto de origen, pulsar sobre , y posteriormente seleccionar el punto en planta.
  • Existe la posibilidad de mantener las plataformas conectadas entre la arista común, activando la casilla [ ] Ajustar plataformas conectadas.
  • Para ver los cambios realizados se puede:
    • Pulsar sobre el icono  (pestaña Plataformas).
    • ir a pestaña Posición, borrar (dibujos) y volver a Generar.

De forma global también se pueden cambiar los valores de las pendientes desde el apartado Pendientes Globales WTG Actual. Estos cambios afectan a todas las subexplanadas:




Las acciones posibles son:

  • Pendiente longitudinal WTG:
    • Seleccionar el Punto de giro:
      • Coordenadas WTG.
      • Seleccionable por el usuario con .
    • Pendiente longitudinal a aplicar (en %).
  • Pendiente transversal WTG. Tiene 2 modos de trabajo:
    • Lado WTG: Se abaten todas las subexplanadas hacia un lado.
      • Punto de giro: Marca el punto sobre el que se abate. El punto de giro puede ser:
        • Coordenadas del WTG.
        • Borde del eje.
        • Seleccionable por el usuario con .
      • Pendiente transversal a aplicar (en %).
    • Ambos lados: Se abate a los 2 lados del vial de forma independiente.
      • La forma de proceder es similar a Lado WTG, pero ahora a ambos lados.
  • Reiniciar valores a 0: Se vuelve a pendiente 0 y a la cota correspondiente a Z mínima de la pestaña Posición.


Perfiles

En esta pestaña se generan.
  • Archivos de perfiles *.per sobre las líneas directrices definidas en la pestaña Tipos.
  • Rotulación en planta de las líneas directrices definidas en la pestaña Tipos.

Para la generación de perfiles *.per se configuran las siguientes opciones:

  • Número de ficheros a generar:
    • Archivo *.per único para todos los perfiles: Si está marcado, genera un único archivo *.per para todos los perfiles.
    • Archivo *.per para cada WTG: Si está marcado, se genera un *.per para cada posición de los aerogeneradores. 
  • [] Todos:
    • Si está activa, se generan los perfiles de todas las posiciones de los aerogeneradores.
    • Si está desactiva, se generan los perfiles de las posiciones indicadas en Desde WTG, hasta a WTG.
  • Nombre: Nombre del archivos o de los archivos *.per generados.
  • Semiancho de banda: Semiancho de banda a considerar para el cálculo de la generación de los perfiles.
  • Generar: Botón para generar los archivos *.per.

Para la rotulación se configuran las siguientes opciones:

  • [] Todos:
    • Si está activa, se rotulan en todas las posiciones de los aerogeneradores.
    • Si está desactiva, se rotulan en las posiciones indicadas en las casillas Desde WTG hasta a WTG.
  • [ ] Marcas extremos: Rotula también los extremos.
  • Para la rotulación hay 2 filas de datos, cada fila de datos se corresponde con una rotulación en PKs según cierta equidistancia.



    • [ ]: Si esta activa de activación que aparece en primer lugar está activa, rotula según dicha fila de datos.
    • Equid.: Equidistancia a rotular.
    • Dis. eje: Distancia a la línea directriz para la rotulación de los símbolos empleados S35 y S36 por defecto.
    • Símbolo Pk
      • Símbolo S35 por defecto: símbolo empleado para los pk sin decimales.
      • Símbolo S36 por defecto: símbolo empleado para los pk con decimales.
    • Dis. eje: Distancia a la línea directriz para la rotulación de la marca empleada, por defecto S11.
    • Marca (90º): Símbolo S11 por defecto empleado para representar una marca en la línea directriz a la equidistancia definida.
  • Generar rótulos: Botón de acción para rotular en planta las líneas directriz
  • : Borra las rotulaciones.
