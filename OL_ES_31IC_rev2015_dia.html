# D1 / D2 / D3 / D4 öğelerinin anlamlarının açıklaması                                       #
# (1) Geçiş durumunda, bir geçiş kesiminin minimum uzunluğunu ifade eder       #
# (2) Gözlemci, giriş ekseni ile kavşak ekseni arasındaki bağlantıdan D1 mesafesi kadar uzakta bir noktaya konumlandırılacaktır.   #
#     D1 durma mesafesi DP veya sabit bir değer olabilir.                         #
#     D2, eksen1-eksen2 bağlantı noktasından başlayarak kavşak ekseni üzerinde ölçülen uzunluktur.      #
#     EĞER D2 = 999 ise, giriş ve kavşak eksenlerinin 1 ve 2 noktaları arasındaki teğet noktası hesaplanacak      #
#     ve MESAFE değerine göre kaydırılacaktır.                                               #
# (3) Bu durumda D1 için DP veya sabit bir değer de belirtilebilir.                             #
# (4) D1 (1,2,3), kaplama kenarlarının (kod 1, 2 veya her ikisi) nasıl inceleneceğini belirtir.           #
# (5) Bağlantılar durumunda D1 (değerler 0-4), ana yol ile kol arasındaki bağlantı noktasını (ABCDE) seçmek için kullanılır.  #
# Ayrıca D3 ve D4'ü uygulamak için D2 mesafesi ayarlanır ve çalışma yapılacak ana yol ve koldaki noktalar bulunur.   #
# DP kullanılabilir.                  #
#(6) İşaretleme durumunda, D1 + D2 toplamından daha az veya eşit bir mesafede bulunan semboller incelenecektir.  #

# GEÇİŞ İÇİN DÜŞEY KURP : Sadece Tepe Kurpları ve Tek Kaplamalı Yollar İçin      #
# Geçiş mesafesi kullanılır ve formülde yerine konur           #
# Engel H2'si 1.1 ile değiştirilir                                                         #
####################################################################################
# KA0 : MİNİMUM DÜŞEY KURPLAR (U. KURPLAR) : FORMÜLLERE GÖRE :                         #
#----------------------------------------------------------------------------------#
# DERE DÜŞEY KURBU: Kv= (Dp*Dp) / ( 2 * ( HF - H2 + Dp*Tg(ALF)) )                   #
# TEPE DÜŞEY KURBU: KV= (Dp*Dp) / ( 2 * (sqrt(H1) + sqrt(H2))^2 )                   #
# Burada Dp = (V * Tp/3.6) + (V*V / (254*(Fl + i)) )                               #
# Fl tablodan hıza göre alınarak ve i=0 yapılarak (Yönetmeliğe göre)         #
#----------------------------------------------------------------------------------#
  KA0U   1           : 0->Bu formülleri kullanma  1->Bu formülleri kullan           #
####################################################################################
# KA1 : MİNİMUM DÜŞEY KURPLAR (K. KURPLAR) : UZUNLUK KONTROLÜ  L >=  KA1 * Vp      #
#----------------------------------------------------------------------------------#
# Düşey kurp uzunluğu KA1 * V'den küçükse, bu değeri karşılayana kadar KV artırılır:   #
# KV = L / A = KA1 * Vp/ A   burada:                          #
# KA1: yapılandırılabilir bir katsayı, İspanyol yönetmeliğine göre 1.00                   #
#   A: Eğimler arasındaki farkın mutlak değeri, ondalık olarak             #
#  Vp: Proje Hızı                                                       #
# 5.3.2.2 Sayfa 65
#----------------------------------------------------------------------------------#
  KA1U   1           : 0->Bu formülleri kullanma  1->Bu formülleri kullan           #
  KA1    1.00        :   1.00 İspanyol Yönetmeliğine göre                                 #
####################################################################################
# KA2 : MİNİMUM DÜŞEY KURPLAR (K. KURPLAR) : UZUNLUK KONTROLÜ > Dp                #
#----------------------------------------------------------------------------------#
# Düşey kurp uzunluğu Dp'den küçükse, kısa düşey kurplar için formüller kullanılır:        #
#                                                                 #
# DERE DÜŞEY KURBU: KV= 2*Dp/A + (2 * ( HF - H2 + Dp*Tg(ALF)) ) / (A*A)             #
# TEPE DÜŞEY KURBU: KV= 2*Dp/A + (2 * (sqrt(H1) + sqrt(H2))^2 ) / (A*A)             #
# Burada :                                                                         #
#   A: Eğimler arasındaki farkın mutlak değeri, ondalık olarak             #
#----------------------------------------------------------------------------------#
  KA2U   1           : 0->Bu formülleri kullanma  1->Bu formülleri kullan           #
